What is Firebase?

Firebase is Google's backend-as-a-service platform that allows you to build web applications without managing servers. In this tutorial, we'll use Firebase Realtime Database to create a simple poll that updates in real-time across all users.

How It Works

This app demonstrates several key Firebase concepts:

1. Firebase Configuration

We connect to Firebase using configuration settings in our JavaScript file.

2. Real-time Database

When someone votes, the data is saved to Firebase and automatically updates on all connected devices.

3. Event Listeners

Firebase listens for changes in the database and updates our UI automatically.
